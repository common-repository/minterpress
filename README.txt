Minterpress lets creators and collectors mint and host eco-friendly NFTs on Wordpress.

== Description ==

Minterpress is a new kind of NFT tool that lets creators and collectors mint and host eco-friendly NFTs on Wordpress. Built on the Tezos blockchain, NFTs minted through Minterpress are energy-efficient, without the negative downsides of first generation NFTs.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow set up instructions within the plugin.
4. Make sure you have permalinks enabled.
